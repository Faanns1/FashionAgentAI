## 📌 Deskripsi Proyek

FashionAgentAI merupakan aplikasi rekomendasi fashion berbasis **Agentic AI** yang dirancang untuk membantu pengguna menemukan pakaian yang sesuai dengan preferensi, gaya, dan kebutuhan mereka. Sistem ini memanfaatkan beberapa agen AI yang bekerja secara kolaboratif untuk menganalisis permintaan pengguna, mencari produk fashion yang relevan, serta menghasilkan rekomendasi dalam bentuk teks dan gambar.

## ✨ Fitur Utama

* 🤖 Implementasi arsitektur Multi-Agent AI.
* 👕 Rekomendasi outfit yang dipersonalisasi sesuai preferensi pengguna.
* 🔍 Pencarian produk fashion secara cerdas.
* 🖼️ Menampilkan rekomendasi dalam bentuk gambar dan deskripsi.
* 💬 Penjelasan rekomendasi yang dihasilkan oleh AI.
* 🎯 Analisis preferensi dan kebutuhan pengguna.

## 🛠️ Teknologi yang Digunakan

* Python
* Streamlit
* Agentic AI
* Large Language Model (LLM)
* HTML, CSS, dan JavaScript
* Git & GitHub

## 🎯 Tujuan Proyek

Proyek ini dikembangkan sebagai implementasi konsep **Agentic AI** dalam bidang rekomendasi fashion. Tujuannya adalah membangun sistem yang mampu memberikan rekomendasi pakaian secara cerdas, personal, dan interaktif melalui kolaborasi beberapa agen AI.
